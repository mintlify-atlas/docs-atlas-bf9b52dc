  <h2 className="text-3xl font-semibold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get CandyShell up and running in minutes
  </p>
  
  <Steps>
    <Step title="Install dependencies">
      CandyShell requires LuaJIT, GTK4, Adwaita, and Gtk4-Layer-Shell with lgi bindings.
      
      ```bash
      # On Arch Linux
      sudo pacman -S luajit gtk4 libadwaita gtk4-layer-shell lua-lgi
      
      # On Ubuntu/Debian (may need PPA for latest GTK4)
      sudo apt install luajit libgtk-4-dev libadwaita-1-dev gtk4-layer-shell
      ```
    </Step>
    
    <Step title="Clone the repository">
      Get the CandyShell source code from GitHub.
      
      ```bash
      git clone https://github.com/poloz-z/candy-shell.git
      cd candy-shell
      ```
    </Step>
    
    <Step title="Run CandyShell">
      Launch CandyShell with LuaJIT. Make sure you're running a Wayland compositor like Sway.
      
      ```bash
      luajit main.lua
      ```
      
      <Note>
        CandyShell is designed to work with the Sway window manager for gap control and workspace management.
      </Note>
    </Step>
  </Steps>
</div>
