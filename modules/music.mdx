---
title: Music
description: MPRIS media player integration for controlling and monitoring playback
---

The `music` module provides integration with MPRIS-compatible media players, allowing you to control playback, retrieve track information, and extract album artwork. It uses Playerctl to communicate with any media player that supports the MPRIS D-Bus interface.

## Usage

```lua
local music = require("libs.music")

-- Get current track info
local info = music.player_info()
print("Now playing: " .. info.artist .. " - " .. info.title)

-- Control playback
music.play_pause()
music.next()
music.previous()

-- Get album artwork
local pixbuf = music.get_album_art()
```

## Functions

### player_info

```lua
local info = music.player_info()
```

Retrieves comprehensive information about the currently playing track.

**Returns:** Table with track metadata and playback status

```lua
-- When a player is active
{
  artist = "Daft Punk",                        -- Track artist
  title = "Get Lucky",                         -- Track title
  album = "Random Access Memories",            -- Album name
  artUrl = "file:///path/to/cover.jpg",       -- Album art URL
  length = 367000000,                          -- Track length in microseconds
  status = "Playing"                           -- Playback status
}

-- When no player is available
{
  artist = "none",
  title = "none",
  album = "none",
  artUrl = "none",
  length = 0,
  status = "EXITED"
}
```

**Playback Status Values:**
- `"Playing"` - Currently playing
- `"Paused"` - Playback paused
- `"Stopped"` - Playback stopped
- `"EXITED"` - No player available

**Example:**

```lua
local music = require("libs.music")
local info = music.player_info()

if info.status ~= "EXITED" then
  print("Now playing:")
  print("  Artist: " .. info.artist)
  print("  Title: " .. info.title)
  print("  Album: " .. info.album)
  print("  Status: " .. info.status)
else
  print("No media player active")
end
```

### length

```lua
local seconds = music.length()
```

Returns the total length of the current track in seconds.

**Returns:** Track length in seconds (integer), or 0 if unavailable

**Example:**

```lua
local len = music.length()
local minutes = math.floor(len / 60)
local seconds = len % 60
print(string.format("Duration: %d:%02d", minutes, seconds))
```

### get_position

```lua
local seconds = music.get_position()
```

Returns the current playback position in seconds.

**Returns:** Current position in seconds (integer), or 0 if unavailable

**Example:**

```lua
local pos = music.get_position()
local len = music.length()
local progress = (pos / len) * 100
print(string.format("Progress: %.1f%%", progress))
```

### get_album_art

```lua
local pixbuf = music.get_album_art()
```

Retrieves album artwork as a GdkPixbuf. Supports both base64-encoded and file:// URLs.

**Returns:** `GdkPixbuf.Pixbuf` object, or `nil` if no artwork is available

**Supported Formats:**
- Base64 data URIs: `data:image/png;base64,...`
- File URIs: `file:///path/to/cover.jpg`

**Example:**

```lua
local Gtk = require("lgi").require("Gtk", "4.0")
local music = require("libs.music")

local album_art_image = Gtk.Image.new()

local function update_album_art()
  local pixbuf = music.get_album_art()
  
  if pixbuf then
    -- Scale to desired size
    local scaled = pixbuf:scale_simple(200, 200, "BILINEAR")
    album_art_image:set_from_pixbuf(scaled)
  else
    -- Use placeholder
    album_art_image:set_from_icon_name("audio-x-generic")
  end
end
```

### get_album_colors

```lua
local bg_color, fg_color = music.get_album_colors()
```

Extracts dominant color from album artwork and calculates a contrasting text color.

**Returns:**
- `bg_color` (string): Background color as hex string (e.g., `"#bb86fc"`)
- `fg_color` (string): Foreground/text color as hex string (`"#000000"` or `"#ffffff"`)

**Default:** Returns `"#353535"` and `"#ffffff"` if no album art is available

**Example:**

```lua
local music = require("libs.music")

local bg, fg = music.get_album_colors()

-- Apply colors to widget via CSS
local css = string.format([[
  .music-widget {
    background-color: %s;
    color: %s;
  }
]], bg, fg)

css_provider:load_from_data(css, #css)
```

### reconnect_player

```lua
local player = music.reconnect_player()
```

Forces reconnection to the media player. Useful when a player exits and restarts.

**Returns:** New player instance or `nil` if no player is available

**Example:**

```lua
local music = require("libs.music")

-- Try to reconnect when player exits
local info = music.player_info()
if info.status == "EXITED" then
  print("Player exited, attempting reconnect...")
  local player = music.reconnect_player()
  
  if player then
    print("Successfully reconnected")
  else
    print("No player available")
  end
end
```

## Complete Music Player Widget

```lua
local music = require("libs.music")
local Gtk = require("lgi").require("Gtk", "4.0")
local GLib = require("lgi").GLib

local function create_music_player()
  local box = Gtk.Box.new(Gtk.Orientation.VERTICAL, 10)
  
  -- Album art
  local album_art = Gtk.Image.new()
  album_art:set_pixel_size(200)
  box:append(album_art)
  
  -- Track info
  local title_label = Gtk.Label.new()
  title_label:add_css_class("title-4")
  box:append(title_label)
  
  local artist_label = Gtk.Label.new()
  artist_label:add_css_class("dim-label")
  box:append(artist_label)
  
  local album_label = Gtk.Label.new()
  album_label:add_css_class("caption")
  box:append(album_label)
  
  -- Progress bar
  local progress = Gtk.ProgressBar.new()
  box:append(progress)
  
  local time_label = Gtk.Label.new()
  box:append(time_label)
  
  -- Controls
  local controls = Gtk.Box.new(Gtk.Orientation.HORIZONTAL, 10)
  controls:set_halign(Gtk.Align.CENTER)
  
  local prev_btn = Gtk.Button.new_from_icon_name("media-skip-backward")
  local play_btn = Gtk.Button.new_from_icon_name("media-playback-start")
  local next_btn = Gtk.Button.new_from_icon_name("media-skip-forward")
  
  controls:append(prev_btn)
  controls:append(play_btn)
  controls:append(next_btn)
  box:append(controls)
  
  -- Control handlers
  prev_btn.on_clicked = function() music.previous() end
  play_btn.on_clicked = function() music.play_pause() end
  next_btn.on_clicked = function() music.next() end
  
  -- Update function
  local function update_player()
    local info = music.player_info()
    
    if info.status == "EXITED" then
      title_label:set_text("No player active")
      artist_label:set_text("")
      album_label:set_text("")
      album_art:set_from_icon_name("audio-x-generic")
      return true
    end
    
    -- Update info
    title_label:set_text(info.title)
    artist_label:set_text(info.artist)
    album_label:set_text(info.album)
    
    -- Update album art
    local pixbuf = music.get_album_art()
    if pixbuf then
      local scaled = pixbuf:scale_simple(200, 200, "BILINEAR")
      album_art:set_from_pixbuf(scaled)
      
      -- Apply album colors
      local bg, fg = music.get_album_colors()
      box:set_css_classes({"music-widget"})
    else
      album_art:set_from_icon_name("audio-x-generic")
    end
    
    -- Update play/pause button
    if info.status == "Playing" then
      play_btn:set_icon_name("media-playback-pause")
    else
      play_btn:set_icon_name("media-playback-start")
    end
    
    -- Update progress
    local pos = music.get_position()
    local len = music.length()
    if len > 0 then
      progress:set_fraction(pos / len)
      time_label:set_text(string.format("%d:%02d / %s",
        math.floor(pos / 60), pos % 60,
        music.length_min_sec()))
    end
    
    return true  -- Continue timer
  end
  
  -- Update every second
  GLib.timeout_add_seconds(GLib.PRIORITY_DEFAULT, 1, update_player)
  update_player()  -- Initial update
  
  return box
end
```

## MPRIS Properties

The module uses standard MPRIS metadata properties:

- `xesam:artist` - Track artist(s)
- `xesam:title` - Track title
- `xesam:album` - Album name
- `mpris:artUrl` - Album artwork URL
- `mpris:length` - Track length in microseconds

## Implementation Details

### Player Lifecycle

The module maintains a singleton player instance that is automatically cleaned up when the player exits:

```lua
-- libs/music.lua:11-24
local function init_player()
  if player then return player end
   
  local new_player = Playerctl.Player.new()
  
  if new_player then
    player = new_player
    player.on_exit = function(self)
      player = nil
      collectgarbage("collect") 
    end
  end
  return player
end
```

### Album Art Decoding

The module handles two types of album art URLs:

1. **Base64 Data URIs:**
```lua
if artUrlstr and string.find(artUrlstr, "^data:image/.*;base64,") then
  local image64 = string.gsub(artUrlstr, "^data:image/[^;]*;base64,", "")
  local decode_image = GLib.base64_decode(image64)
  -- ... load into pixbuf
end
```

2. **File URIs:**
```lua
if artUrlstr and string.find(artUrlstr, "^file://") then
  local file_path = string.gsub(artUrlstr, "^file://", "")
  file_path = GLib.uri_unescape_string(file_path, nil)
  return GdkPixbuf.Pixbuf.new_from_file(file_path)
end
```

### Color Extraction

Dominant color is extracted by scaling the album art to 3x3 pixels and reading the first pixel:

```lua
-- libs/music.lua:151-165
local one_pixel = pixbuf:scale_simple(3, 3, 'BILINEAR')
local bytes = one_pixel:read_pixel_bytes()
local data = bytes:get_data() 

local r = string.byte(data, 1)
local g = string.byte(data, 2)
local b = string.byte(data, 3)

local bg_hex = string.format("#%02x%02x%02x", r, g, b)
```

Text color is calculated based on luminance:

```lua
-- libs/music.lua:139-142
local function get_contrasting_text_color(r, g, b)
  local luminance = (0.299 * r + 0.587 * g + 0.114 * b)
  return luminance > 128 and "#000000" or "#ffffff"
end
```

## Supported Players

Any MPRIS-compatible media player is supported, including:

- Spotify
- VLC
- mpv (with MPRIS script)
- Chromium/Chrome (with media playing)
- Firefox (with media playing)

- Audacious
